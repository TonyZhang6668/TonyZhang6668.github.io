# Academic Personal Website

Static academic personal website built with Astro, TypeScript, and plain CSS.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Personalization

Most content lives in `src/data/site.ts`. Replace placeholder profile details, research areas, experiences, projects, news, awards, and service entries there. Replace assets in `public/` with your real headshot, CV, project thumbnails, favicon, and social preview image.

For GitHub Pages project sites, set repository variables:

- `SITE_URL`: your final site URL, for example `https://username.github.io`
- `SITE_BASE`: `/repo-name/` for project pages, or `/` for `username.github.io`
